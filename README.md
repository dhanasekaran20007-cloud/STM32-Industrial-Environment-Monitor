# STM32 Industrial Environment Monitor

## 📌 Overview

This project implements an Industrial Environment Monitoring System using the STM32F103C8T6 Blue Pill.

The system continuously monitors environmental parameters such as temperature, humidity, and light intensity. It displays real-time data on an LCD, provides alarm indications, and is designed using modular embedded software architecture.

This project is being developed incrementally to learn professional embedded systems development.

---

## 🎯 Objectives

- Learn STM32 peripheral programming
- Develop reusable embedded drivers
- Interface multiple sensors
- Display real-time environmental data
- Implement alarm mechanisms
- Integrate FreeRTOS
- Follow modular software architecture
